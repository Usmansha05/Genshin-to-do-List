🌸 Genshin To-Do List
A sleek, fan-made to-do list app designed specifically for Genshin Impact players. Track your daily commissions, resin management, weekly bosses, event tasks, and farming goals — all in one place.

✨ Features
✅ Daily Tracker: Commissions, domains, resin recovery, and expeditions.
🗓️ Weekly Tracker: Bosses, talent books, weapon ascension materials.
📆 Custom Events: Create and track personal goals or ongoing events.
🔔 Reminders: Optional notifications for resin full, reset timers, and more.
🎨 Themed UI: Beautiful Genshin-style interface with responsive design.
☁️ Data Sync (optional): Store tasks locally or sync using cloud/database.
